To use Google account integration, please configure your Google Console (https://console.cloud.google.com) with the following:

  1. Create an OAuth Client ID under Credentials
  2. Set the following scopes:
    • https://www.googleapis.com/auth/user.addresses.read
    • https://www.googleapis.com/auth/user.phonenumbers.read
    • https://www.googleapis.com/auth/userinfo.email
    • profile
  3. Use this fixed redirect URI:
    https://client-management.online/oauth2callback
